# Funcionalidades de Ryuu Fit

A continuación se presenta una lista de las funcionalidades disponibles para el usuario de la aplicación:

## 1. Perfil Inicial
El usuario completa un cuestionario breve sobre complexión, edad, altura, nivel de actividad física y objetivos personales (pérdida de peso, tonificación, ganancia muscular, etc.).  
También puede repetir el cuestionario cuando lo desee.

## 2. Rutinas de Entrenamiento en Casa
- Rutinas diseñadas para realizarse sin equipo profesional.  
- Explicación de qué área del cuerpo se trabaja en cada ejercicio.  
- Imágenes demostrativas de los ejercicios.  
- Temporizador regresivo integrado para gestionar descansos y series.  
- Adaptadas al espacio disponible en el hogar.

## 3. Rutinas y Ejercicios por Objetivo
Rutinas enfocadas según la elección del usuario: **Cardio, Fuerza, Resistencia y Elasticidad**.

## 4. Plan de Alimentación Básico
- Sugerencias de dietas equilibradas generales.  
- Opciones saludables para cada comida del día con imágenes de referencia.  
- Adaptadas a hábitos, recursos y preferencias alimenticias, buscando equilibrio entre salud y accesibilidad.

## 5. Accesibilidad y Comodidad
Convierte cualquier espacio disponible en un gimnasio personal, eliminando barreras de tiempo, ubicación o equipo.  
El diseño es simple e intuitivo para que cualquier usuario pueda comenzar fácilmente.
